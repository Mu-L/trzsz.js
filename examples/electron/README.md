# Electron example for trzsz
*Under development ...*

## Getting Started

* Start test app
```sh
cd /path/to/examples/electron
cd ../.. && npm install && npm run build && cd -
npm install
npm start
```

* Install trzsz server
```sh
sudo python -m pip install --upgrade trzsz-libs trzsz-svr
```

* Upload files
```sh
trz
```

* Download files
```sh
tsz file1 file2
```

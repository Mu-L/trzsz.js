# TrzszAddon example for trzsz
*Under development ...*

## Getting Started

* Start test server
```sh
cd /path/to/examples/addon
cd ../.. && npm install && npm run build && cd -
npm install
npm start
```

* Open web browser
```
http://localhost:8081
```

* Install trzsz server
```sh
sudo python -m pip install --upgrade trzsz-libs trzsz-svr
```

* Upload files
```sh
trz
```

* Download files
```sh
tsz file1 file2
```
